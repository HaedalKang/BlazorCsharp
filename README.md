# BlazorCsharp
C# Blazor로 배우는 프론트엔드의 기초

# Goal
프론트엔드의 기초를 이해하고 Blazor를 이용하여 웹 어플리케이션을 만드는 실습을 한다.

# Target
 - 프론트엔드의 기초를 알고 싶은 사람
 - Blazor의 사용법을 배우고 싶은 사람
 - Microsoft Azure를 통해 Blazor로 만든 웹앱을 배포하고 싶은 사람
